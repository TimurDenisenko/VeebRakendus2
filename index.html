Abi
1. Kui hõljutate kursorit esimese teabega elemendi kohal, muutub selle tagavärv kollaseks ja äärised oranžiks
2. Kui klõpsate esimesel teabega elemendil, ilmub piltidega element
3. Kui klõpsate teisel piltidega elemendil, suletakse piltidega element
4. Kui hõljutate kursorit piltide kohal, muutub nende suurus
5. Kui klõpsate 3. elemendil ja kursor on sellel, hakkab see liikuma paremasse alumisse nurka ja ka initsiaalid, aga kui kursor on väljaspool kolmanda elemendi piirkonda, siis see ja initsiaalid naasevad
6. initsiaale saab liigutada
7. Kui hõljutate kursorit selle elemendi kohal, kus on tekst abi kohta, hakkab see vilkuma
